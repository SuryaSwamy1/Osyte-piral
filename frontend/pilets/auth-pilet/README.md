# Auth Pilet

Professional authentication pilet with login and signup pages.

## Features

- Professional login page
- Professional signup page
- JWT token-based authentication
- Form validation
- Error handling
- Responsive design
- Loading states

## Setup

```bash
cd pilets/auth-pilet
npm install
```

## Development

```bash
npm start
```

## Build

```bash
npm build
```

## Usage

The pilet registers two pages:
- `/login` - Login page
- `/signup` - Signup page

These pages integrate with the backend API for authentication.

## API Integration

The pilet connects to the backend API at `http://localhost:3001/api/auth`

Make sure the backend server is running before using this pilet.
